## Prerequisities
* Python 3
* Java 9
* org.json
  
## Build
  * Run program with: `make FILE=<expand_file> FILE2=<json_file>`
    * e.g. `make FILE=fib/fib.c.234r.expand FILE2=fib/fib.json`
  * Clean up with: `make clean`
